# MLB Rivalry Tracker: Yankees vs Red Sox

This project uses the `mlbstatsapi` to analyze the historic rivalry between the Yankees and Red Sox from 2000 to 2023. It pulls game data, tracks wins, and visualizes trends.

## Features
- Pulls game data using MLB Stats API
- Analyzes win/loss trends
- Visualizes rivalry dynamics

## How to Run
```bash
pip install -r requirements.txt
python rivalry_tracker.py
```
